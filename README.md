# sass-starter-package
starter package to start with that covert sass to css and others

# what are included:
  
1. **autoprefixer**

2. **bootstrap**

3. **live-server**

4. **node-sass**

5. **nodemon**

6. **npm-run-all**

7. **postcss**

8. **postcss-cli**

9. **purgecss**

10. **sass**

11. **stylelint**

12. **stylelint-config-twbs-bootstrap**

# How to start

* npm i
* npm start
